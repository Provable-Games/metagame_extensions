# entry_validator_test_common

Shared test infrastructure for entry validator tests.

## Contents

- `constants` - Mainnet/Sepolia addresses for tournament platforms (e.g. Budokan), tokens, governance contracts
- `mocks/entry_validator_mock` - Token-gated mock validator for unit tests
- `mocks/open_entry_validator_mock` - Open (no-gate) mock validator for unit tests
