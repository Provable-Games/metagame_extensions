# Examples

Uncompiled reference and work-in-progress validator implementations. These files are not part of any Scarb package and serve as templates or documentation only.

- `adventurer_validator.cairo` - Validator based on Loot Survivor adventurer ownership
- `beast_validator.cairo` - Validator based on Beast NFT ownership
- `snapshot_ETHEREUM_validator.cairo` - Cross-chain snapshot validator (Ethereum L1)
- `tournament_validator_v2.cairo` - V2 tournament qualifier with expanded modes
